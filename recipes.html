規格外のトマトで作るトマトスープ

※写真はイメージです。

材料：

  • 規格外のトマト...4個
  • 玉ねぎ...1個
  • にんにく...1かけ
  • オリーブオイル...大さじ1
  • 水...500ml
  • コンソメキューブ...1個
  • 塩、こしょう...適量

作り方：

  1. トマトを一口大に切る。
  2. 玉ねぎとにんにくをみじん切りにする。
  3. 鍋にオリーブオイルを熱し、にんにくを入れて香りを出す。
  4. 玉ねぎを加え、しんなりするまで炒める。
  5. トマトを加えて炒める。

規格外さつまいもの天ぷら

※写真はイメージです。

材料：

  • さつまいもの葉っぱ 数枚
  • 天ぷら粉 適量
  • 水 適量
  • 揚げ油 適量

作り方：

  1. さつまいもを洗って水気を切り、食べやすい大きさに切る。
  2. 天ぷら粉と水を混ぜ、固さを調整する。
  3. 揚げ油を熱し、2の生地に1のサツマイモをくぐらせて揚げる。
  4. カリッと揚がったら、取り出して油を切る。
  5. お好みで塩やタレをつけて食べる。

規格外野菜のピクルス

※写真はイメージです。

材料：

  • 野菜（キャベツ、人参、きゅうりなど） 500g
  • 酢 300ml
  • 砂糖 200g
  • 塩 20g
  • 薬味（にんにく、唐辛子、生姜など） 適量

作り方：

  1. 野菜を切って適当な大きさに切ります。
  2. 鍋に酢、砂糖、塩を入れて火にかけ、砂糖と塩が溶けるまで混ぜます。
  3. 火を止めて、薬味を入れ、冷めるまで放置します。
  4. 野菜を瓶に詰め、2で作った液体を加えます。
  5. 冷蔵庫で1日以上寝かせたら完成です。

規格外野菜のポテトサラダ

※写真はイメージです。

材料：

  • じゃがいも 3個
  • にんじん 1本
  • 玉ねぎ 1個
  • ツナ缶 1缶
  • マヨネーズ 大さじ3
  • 塩、こしょう 適量

作り方：

  1. じゃがいもとにんじんを茹でて、皮を剥き、一口大に切ります。
  2. 玉ねぎはみじん切りにし、水に10分ほどさらして辛みを抜きます。
  3. ツナ缶を汁ごと缶から出し、ボウルに入れます。
  4. 2に塩、こしょうを加えて揉み込みます。
  5. 1のじゃがいも、にんじん、3のツナをボウルに入れ、マヨネーズを加えて混ぜます。
  6. 好みに合わせて味を調整し、完成です。

規格外人参のスープ

※写真はイメージです。

材料：

  • 規格外人参 500g
  • 玉ねぎ 1個
  • セロリ 2本
  • ガーリック 2かけ
  • オリーブオイル 大さじ2
  • 水 1リットル
  • コンソメキューブ 1個
  • 塩 適量
  • ピンクペッパー 適量

作り方：

  1. 規格外人参は皮をむいて1cm程度の薄切りに、玉ねぎとセロリは粗く刻む。
  2. 鍋にオリーブオイルを熱し、刻んだ野菜を加えて炒める。
  3. ガーリックを加え、香りが立ったら水を加え、コンソメキューブを入れる。
  4. 落とし蓋をして弱火で15分ほど煮る。
  5. 火を止め、塩で味を調える。
  6. 皿に盛り、ピンクペッパーを振る。

規格外トマトのパスタ

※写真はイメージです。

材料：

  • 規格外トマト 6個
  • パスタ 150g
  • オリーブオイル 大さじ2
  • ニンニク 1かけ
  • 唐辛子 適量
  • 塩 適量
  • パルメザンチーズ 適量

作り方：

  1. 規格外トマトは皮をむいて種を取り除き、1cm角に切る。
  2. パスタは表示通りに茹でる。
  3. フライパンにオリーブオイルを入れ、ニンニクと唐辛子を加えて香りを出す。
  4. 規格外トマトを加え、弱火で10分ほど煮る。
  5. 塩で味を調え、パスタと絡める。
  6. パルメザンチーズをかけて完成。

  • オリーブオイル 適量

  4. キッチンペーパーなどで余分な油を取り除き、塩をふって完成です。

  5. フードプロセッサーなどで滑らかになるまで混ぜ、生クリームを加えて混ぜます。
  6. .塩、こしょうで味を整え、器に盛り付けて完成です。

規格外野菜のポテトグラタン

※写真はイメージです。

材料：

  • 規格外のじゃがいも 4個
  • 規格外の玉ねぎ 1個
  • 規格外のにんにく 2かけ
  • 生クリーム 200ml
  • 牛乳 200ml
  • チーズ 適量
  • 塩、コショウ 各適量
  • オリーブオイル・・・適量

作り方：

  1. じゃがいも、玉ねぎ、にんにくをスライスする。
  2. オリーブオイルを熱したフライパンで、玉ねぎとにんにくを炒める。
  3. 炒めた玉ねぎとにんにくを別の器に移す。
  4. 同じフライパンで、じゃがいもを焼き色がつくまで炒める。
  5. 焼き色がついたら、塩、コショウを加え、玉ねぎとにんにくを戻し、混ぜる。
  6. オーブンを200℃に予熱する。
  7. オーブン用の耐熱皿に、じゃがいもの混ぜ物を入れ、生クリームと牛乳を加える。
  8. チーズをかける。
  9. オーブンで30分焼いて完成。

ピリ辛豆腐と規格外野菜の炒め物

※写真はイメージです。

材料：

  • 豆腐 1丁
  • 規格外玉ねぎ 1/2個
  • 規格外赤パプリカ 1個
  • 規格外黄パプリカ 1個
  • 規格外人参 1本
  • 規格外ミニトマト 5個
  • 規格外にんにく 1かけ
  • 規格外しょうが 1かけ
  • 規格外青唐辛子 1本
  • 酒 大さじ2
  • 醤油 大さじ2
  • 砂糖 大さじ1
  • 塩コショウ 適量
  • サラダ油 適量

作り方：

  1. 豆腐を水切りし、2cm角に切る。玉ねぎは縦に薄切り、パプリカは1cm角に切る。人参は斜め薄切り、ミニトマトは半分に切る。にんにくとしょうがはみじん切りにする。青唐辛子は斜め薄切りにする。
  2. フライパンにサラダ油を熱し、にんにく、しょうが、青唐辛子を炒める。
  3. 玉ねぎを加えて炒め、透き通ってきたらパプリカ、人参を加えてさらに炒める。
  4. 豆腐を加えてさっと炒める。酒、醤油、砂糖を加え、さらに炒め合わせる。
  5. 最後にミニトマトを加え、塩コショウで味を調える。

規格外野菜を使ったレシピは、野菜の形や大きさが揃っていなくても、味や栄養価に影響はありません。また、規格外野菜を使うことで、通常の野菜よりもリーズナブルに、かつ独創的な料理を楽しむことができます。
